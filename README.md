# Developer Workstation

## Overview

This repository contains the setup, configuration, documentation, and learning journey for building a modern software development workstation.

The objective is to create a reusable development environment for learning and developing applications using:

* Git & GitHub
* Visual Studio Code
* .NET / C#
* Python
* Node.js
* Docker
* GitHub Spec Kit
* Spec-Driven Development (SDD)
* AI-assisted software development
